# 呼吸机使用常见问题解答

## 为什么机器开机没反应？
- 检查是否通电（电源没有插好或是电源接头没插好）
- 电机损坏（部分机器会出现故障代码）
- 主板损坏（部分机器会出现故障代码）
- 主机进水（部分机器会出现故障代码）

**说明**：遇到上述2、3、4点情况，需将机器寄回总部处理。


## 为什么管路里有水？（湿度大或室温低）
### 原理
属于物理现象，冷热空气接触时发生冷凝，形成冷凝水。

### 解决方案
- 调低湿化器档位
- 提高室内温度（例如开启空调）
- 使用加温管
- 放低机器位置
- 用棉毛类物品将管路包裹保温

### 温馨提示
极端温湿度环境下，即使使用加温管，仍可能存在少量冷凝水，但冷凝现象会比使用普通管路明显减轻。


## 机器进水怎么办？
### 解决方案
1. 立即断开机器电源，禁止再次通电
2. 擦拭干净可见的水分
3. 联系并交给服务商进行专业处理

### 不正确的使用行为
- 装包时未倾倒湿化器内的水
- 移动机器时导致机器翻身
- 用水清洗机身


## 用呼吸机为什么会胃肠胀气或放屁？
### 原理
呼吸机输出的气压进入食道并流入肠胃，导致胃肠胀气；若气体能正常排出，则会表现为放屁。

### 解决方案
- 饭后间隔1小时再使用呼吸机
- 适当调小呼吸机输出压力
- 调整睡眠姿势
- 若使用的是口鼻面罩，可考虑更换为鼻罩
- 若气体无法正常排气，可通过轻柔按摩肠胃部位，促进胃肠道蠕动，加速胀气排出


## 用呼吸机可能会出现以下情况？
| 可能出现的情况 | 原因 | 处理方案 |
|----------------|------|----------|
| 眼睛红肿       | 面罩漏气，气流吹到眼睛 | 调整面罩头带，避免漏气 |
| 面部浮肿       | 面罩头带过紧，压迫面部 | 调整面罩头带松紧度，减少面部压迫 |
| 全身酸痛       | 身体适应新的呼吸方式，属于改变常态后的反应 | 放松身体，尽量保持平时习惯的睡眠姿势 |


## 为什么有时候自动关机失效？
### 原理
自动关机功能的触发条件是“漏气量达到一定阈值”，当漏气量未满足该阈值时，机器不会停止工作。

### 可能原因及处理方案
| 可能原因 | 处理方案 |
|----------|----------|
| 漏气量不足（面罩出风口过小） | 拔下面罩，或更换更大出风口的面罩型号 |
| 面罩/管路出风口被障碍物堵住 | 清除出风口处的障碍物 |
| 部分机型需校准或重新设置 | 按照说明书对机器进行重新设置 |
| 机器故障（传感器失灵） | 联系并交给服务商处理 |


## 为什么用呼吸机后嗜睡症状没有改善？
| 原因 | 处理方案 |
|------|----------|
| 对呼吸机不适应，夜间易醒或睡眠质量仍不足 | 调整心态，逐步适应呼吸机 |
| 存在影响治疗效果的不良习惯（如酗酒、服用特定药物） | 引导健康生活方式，纠正不良习惯 |
| 通气模式或参数设置不合理 | 根据呼吸机记录的数据，调整通气模式及参数 |
| 嗜睡由其他病症导致 | 建议前往医院就诊，排查并治疗其他潜在病症 |


## 有这些问题需要综合治疗？
- **失眠**：前期在进行正压通气治疗的同时，遵医嘱服用失眠药物
- **鼻炎**：前期在进行正压通气治疗的同时，使用鼻炎喷剂缓解症状
- **肥胖低通气**：在进行正压通气治疗的同时，配合减重管理


## 为什么用呼吸机睡不着？
| 原因 | 处理方案 |
|------|----------|
| 合并失眠症 | 需同时处理失眠问题，遵医嘱服药后再使用呼吸机 |
| 心理作用（对新设备不适应，改变常态后心理排斥） | 调整心态，通过反复使用逐步适应呼吸机 |
| 压力设置不当 | - 起始压力过小：气压不足，导致气闷难受，需适当调高起始压力<br>- 起始压力过大：气压过冲，产生不适感，需适当调低起始压力 |


## 为什么用呼吸机半夜容易醒？
| 原因 | 说明 |
|------|------|
| 最大压力过大 | 最大压力设置值过高，夜间压力突然升高时易被吹醒 |
| 最大压力过小 | 最大压力设置值低于实际需求，导致气闷难受而醒 |
| 需求压力与承受压力不匹配 | 自身需要较高的治疗压力，但身体无法承受该压力，易被吹醒 |
| 面罩漏气 | 面罩漏气后，机器会自动进行压力补偿，导致压力升高，从而被吹醒 |


## 为什么呼吸机异响？
### 定义
呼吸机异响指机器运行噪音比参数标准噪音超出10分贝以上的情况。

### 可能原因及说明
1. **漏气导致异响**
   - 机器连接问题：主机与湿化器安装不到位、湿化器盖子未扣紧、面罩佩戴不正确、管路连接松动
   - 配件问题：面罩破裂、管路破裂、面罩硅胶垫未安装到位、面罩排气孔有异物堵塞、消音棉脱落

2. **冷凝水导致异响**
   - 管路内残留冷凝水
   - 面罩排气孔内积聚冷凝水

3. **电机原因导致异响**
   - 机器进水导致电机工作状态异常
   - 电机老化
